CYBER CITY SIM

‚è±Ô∏è BUILD TIME: 6 WEEKS üìÅ LOC: 15,234 üî• DIFFICULTY: ADVANCED üéØ PURPOSE: EXPERIMENTAL

üéØ THE CHALLENGE

Creating a performant 3D city simulation in the browser that maintains 60fps while rendering thousands of particles and dynamic lighting. The main challenge was optimizing WebGL draw calls and managing memory efficiently.

‚öôÔ∏è TECH STACK DEEP DIVE

THREE.JS

Used for 3D rendering, camera controls, and scene management. Implemented custom shaders for neon effects and post-processing.

WEBGL SHADERS

Custom GLSL shaders for particle systems, bloom effects, real-time reflections, and screen-space ambient occlusion.

WEB WORKERS

Offloaded physics calculations and particle updates to prevent main thread blocking, achieving smooth 60fps.

INSTANCED RENDERING

Reduced draw calls by 90% using instanced rendering for identical buildings and props.

üöÄ PERFORMANCE OPTIMIZATIONS

  • Instanced Rendering: 90% reduction in draw calls for identical meshes
  • Level of Detail (LOD): Different detail levels based on camera distance
  • Frustum Culling: Only render objects within camera view
  • Texture Atlas: Single texture for all materials
  • Object Pooling: Reuse particle objects instead of creating/destroying
  • Web Worker Offloading: Physics in separate thread

üí° LESSONS LEARNED

WebGL requires careful memory management and draw call optimization. The biggest performance gains came from:

  1. Using instanced rendering for repeated geometry
  2. Implementing proper LOD systems
  3. Moving expensive calculations to Web Workers
  4. Batching draw calls wherever possible

The project taught me that browser-based 3D can rival native applications when optimized correctly.

    // Cyber City - Main Simulation Class
    // ===================================
    // Author: Kai Shi
    // Date: 2024
    // Description: 3D Cyberpunk City Simulation using Three.js

    class CyberCity {
        constructor(config = {}) {
            this.config = {
                particleCount: 10000,
                buildingCount: 256,
                enableShadows: true,
                enableFog: true,
                enablePostProcessing: true,
                ...config
            };
            
            this.scene = null;
            this.camera = null;
            this.renderer = null;
            this.clock = new THREE.Clock();
            this.mixers = [];
            this.particles = [];
            this.buildings = [];
            
            this.init();
        }
        
        // Initialize the simulation
        init() {
            this.createScene();
            this.createCamera();
            this.createRenderer();
            this.createLights();
            this.createCityGrid();
            this.createParticles();
            this.createSkybox();
            this.setupEventListeners();
            
            // Start animation loop
            this.animate();
            
            console.log('üöÄ Cyber City initialized:', {
                particles: this.config.particleCount,
                buildings: this.config.buildingCount,
                shadows: this.config.enableShadows,
                fog: this.config.enableFog
            });
        }
        
        // Create Three.js scene
        createScene() {
            this.scene = new THREE.Scene();
            
            // Add neon fog effect
            if (this.config.enableFog) {
                this.scene.fog = new THREE.FogExp2(0x000000, 0.002);
            }
            
            // Set background color
            this.scene.background = new THREE.Color(0x0a0a0a);
        }
        
        // Create perspective camera
        createCamera() {
            const aspect = window.innerWidth / window.innerHeight;
            this.camera = new THREE.PerspectiveCamera(75, aspect, 0.1, 1000);
            this.camera.position.set(25, 15, 25);
            this.camera.lookAt(0, 0, 0);
        }
        
        // Create WebGL renderer with optimizations
        createRenderer() {
            this.renderer = new THREE.WebGLRenderer({
                antialias: true,
                alpha: false,
                powerPreference: "high-performance"
            });
            
            this.renderer.setSize(window.innerWidth, window.innerHeight);
            this.renderer.setPixelRatio(Math.min(window.devicePixelRatio, 2));
            
            if (this.config.enableShadows) {
                this.renderer.shadowMap.enabled = true;
                this.renderer.shadowMap.type = THREE.PCFSoftShadowMap;
                this.renderer.shadowMap.autoUpdate = true;
            }
            
            // Enable tone mapping for better colors
            this.renderer.toneMapping = THREE.ACESFilmicToneMapping;
            this.renderer.toneMappingExposure = 1.0;
            
            document.querySelector('.demo-frame').appendChild(this.renderer.domElement);
        }
        
        // Create lighting system
        createLights() {
            // Ambient light for base illumination
            const ambientLight = new THREE.AmbientLight(0xffffff, 0.3);
            this.scene.add(ambientLight);
            
            // Main directional light (simulates sun/moon)
            const directionalLight = new THREE.DirectionalLight(0xff0000, 0.8);
            directionalLight.position.set(10, 20, 5);
            directionalLight.castShadow = true;
            
            // Configure shadow properties
            directionalLight.shadow.mapSize.width = 2048;
            directionalLight.shadow.mapSize.height = 2048;
            directionalLight.shadow.camera.near = 0.5;
            directionalLight.shadow.camera.far = 500;
            directionalLight.shadow.camera.left = -50;
            directionalLight.shadow.camera.right = 50;
            directionalLight.shadow.camera.top = 50;
            directionalLight.shadow.camera.bottom = -50;
            
            this.scene.add(directionalLight);
            
            // Neon point lights for buildings
            for (let i = 0; i < 8; i++) {
                const light = new THREE.PointLight(0xff0000, 0.5, 50);
                light.position.set(
                    (Math.random() - 0.5) * 40,
                    5 + Math.random() * 10,
                    (Math.random() - 0.5) * 40
                );
                this.scene.add(light);
            }
        }
        
        // Generate procedural city grid
        createCityGrid() {
            const rows = Math.sqrt(this.config.buildingCount);
            const cols = rows;
            
            // Building geometry (instanced for performance)
            const buildingGeometry = new THREE.BoxGeometry(1, 1, 1);
            const neonMaterial = new THREE.MeshStandardMaterial({
                color: 0xff0000,
                emissive: 0xff0000,
                emissiveIntensity: 0.3,
                metalness: 0.8,
                roughness: 0.2
            });
            
            // Create instanced mesh for optimal performance
            const instancedMesh = new THREE.InstancedMesh(
                buildingGeometry, 
                neonMaterial, 
                this.config.buildingCount
            );
            
            const dummy = new THREE.Object3D();
            
            for (let i = 0; i < this.config.buildingCount; i++) {
                const row = Math.floor(i / cols);
                const col = i % cols;
                
                const x = (col - cols/2) * 2.5;
                const z = (row - rows/2) * 2.5;
                
                // Skip some positions for streets
                if (Math.random() > 0.3) {
                    dummy.position.set(x, 0, z);
                    dummy.scale.set(1, 2 + Math.random() * 4, 1);
                    dummy.rotation.y = Math.random() * Math.PI;
                    dummy.updateMatrix();
                    
                    instancedMesh.setMatrixAt(i, dummy.matrix);
                    
                    this.buildings.push({
                        position: { x, y: dummy.scale.y / 2, z },
                        height: dummy.scale.y,
                        rotation: dummy.rotation.y
                    });
                }
            }
            
            instancedMesh.instanceMatrix.needsUpdate = true;
            instancedMesh.castShadow = true;
            instancedMesh.receiveShadow = true;
            
            this.scene.add(instancedMesh);
            this.instancedBuildings = instancedMesh;
        }
        
        // Create particle system
        createParticles() {
            const particleGeometry = new THREE.BufferGeometry();
            const positions = new Float32Array(this.config.particleCount * 3);
            const colors = new Float32Array(this.config.particleCount * 3);
            const sizes = new Float32Array(this.config.particleCount);
            
            // Initialize particles in random positions
            for (let i = 0; i < this.config.particleCount; i++) {
                // Position
                positions[i * 3] = (Math.random() - 0.5) * 100; // X
                positions[i * 3 + 1] = Math.random() * 50;       // Y
                positions[i * 3 + 2] = (Math.random() - 0.5) * 100; // Z
                
                // Color (neon red with variations)
                colors[i * 3] = 0.8 + Math.random() * 0.2;     // R
                colors[i * 3 + 1] = Math.random() * 0.2;       // G
                colors[i * 3 + 2] = Math.random() * 0.2;       // B
                
                // Size
                sizes[i] = 0.05 + Math.random() * 0.1;
            }
            
            particleGeometry.setAttribute('position', new THREE.BufferAttribute(positions, 3));
            particleGeometry.setAttribute('color', new THREE.BufferAttribute(colors, 3));
            particleGeometry.setAttribute('size', new THREE.BufferAttribute(sizes, 1));
            
            // Custom shader material for particles
            const particleMaterial = new THREE.ShaderMaterial({
                uniforms: {
                    time: { value: 0.0 },
                    pointTexture: { 
                        value: this.createParticleTexture() 
                    }
                },
                vertexShader: `
                    attribute float size;
                    attribute vec3 color;
                    varying vec3 vColor;
                    
                    void main() {
                        vColor = color;
                        vec4 mvPosition = modelViewMatrix * vec4(position, 1.0);
                        gl_PointSize = size * (300.0 / -mvPosition.z);
                        gl_Position = projectionMatrix * mvPosition;
                    }
                `,
                fragmentShader: `
                    uniform sampler2D pointTexture;
                    varying vec3 vColor;
                    
                    void main() {
                        vec4 texColor = texture2D(pointTexture, gl_PointCoord);
                        gl_FragColor = vec4(vColor, texColor.a);
                    }
                `,
                blending: THREE.AdditiveBlending,
                depthTest: false,
                transparent: true,
                vertexColors: true
            });
            
            this.particleSystem = new THREE.Points(particleGeometry, particleMaterial);
            this.scene.add(this.particleSystem);
            
            // Store reference for animation
            this.particlePositions = positions;
            this.particleColors = colors;
        }
        
        // Create particle texture (circular glow)
        createParticleTexture() {
            const canvas = document.createElement('canvas');
            canvas.width = 32;
            canvas.height = 32;
            
            const context = canvas.getContext('2d');
            const gradient = context.createRadialGradient(16, 16, 0, 16, 16, 16);
            
            gradient.addColorStop(0, 'rgba(255, 255, 255, 1)');
            gradient.addColorStop(0.2, 'rgba(255, 100, 100, 0.8)');
            gradient.addColorStop(1, 'rgba(255, 0, 0, 0)');
            
            context.fillStyle = gradient;
            context.fillRect(0, 0, 32, 32);
            
            const texture = new THREE.CanvasTexture(canvas);
            texture.needsUpdate = true;
            
            return texture;
        }
        
        // Create skybox/dome
        createSkybox() {
            const skyGeometry = new THREE.SphereGeometry(200, 32, 32);
            const skyMaterial = new THREE.MeshBasicMaterial({
                color: 0x000000,
                side: THREE.BackSide
            });
            
            const skybox = new THREE.Mesh(skyGeometry, skyMaterial);
            this.scene.add(skybox);
        }
        
        // Setup event listeners
        setupEventListeners() {
            // Window resize
            window.addEventListener('resize', () => this.onWindowResize());
            
            // Mouse movement for camera
            let mouseX = 0, mouseY = 0;
            window.addEventListener('mousemove', (event) => {
                mouseX = (event.clientX / window.innerWidth) * 2 - 1;
                mouseY = -(event.clientY / window.innerHeight) * 2 + 1;
            });
            
            // Store mouse position for animation
            this.mousePosition = { x: 0, y: 0 };
            setInterval(() => {
                this.mousePosition.x += (mouseX - this.mousePosition.x) * 0.05;
                this.mousePosition.y += (mouseY - this.mousePosition.y) * 0.05;
            }, 16);
        }
        
        // Handle window resize
        onWindowResize() {
            this.camera.aspect = window.innerWidth / window.innerHeight;
            this.camera.updateProjectionMatrix();
            this.renderer.setSize(window.innerWidth, window.innerHeight);
        }
        
        // Animation loop
        animate() {
            requestAnimationFrame(() => this.animate());
            
            const delta = this.clock.getDelta();
            const time = this.clock.getElapsedTime();
            
            // Update particle system
            this.updateParticles(delta, time);
            
            // Animate buildings (subtle movement)
            this.animateBuildings(delta, time);
            
            // Update camera based on mouse
            this.updateCamera(time);
            
            // Update particle material time uniform
            if (this.particleSystem.material.uniforms.time) {
                this.particleSystem.material.uniforms.time.value = time;
            }
            
            // Render scene
            this.renderer.render(this.scene, this.camera);
            
            // Update stats if available
            this.updateStats(delta);
        }
        
        // Update particle positions
        updateParticles(delta, time) {
            if (!this.particlePositions) return;
            
            const positions = this.particlePositions;
            
            for (let i = 0; i < positions.length; i += 3) {
                // Move particles upward
                positions[i + 1] += delta * 5;
                
                // Reset particles that go too high
                if (positions[i + 1] > 50) {
                    positions[i + 1] = 0;
                    positions[i] = (Math.random() - 0.5) * 100;
                    positions[i + 2] = (Math.random() - 0.5) * 100;
                }
                
                // Add subtle horizontal movement
                positions[i] += Math.sin(time + i * 0.01) * 0.02;
                positions[i + 2] += Math.cos(time + i * 0.01) * 0.02;
            }
            
            this.particleSystem.geometry.attributes.position.needsUpdate = true;
        }
        
        // Animate buildings
        animateBuildings(delta, time) {
            if (!this.instancedBuildings) return;
            
            const dummy = new THREE.Object3D();
            const matrix = new THREE.Matrix4();
            
            // Rotate instanced buildings slightly
            for (let i = 0; i < this.buildings.length; i++) {
                const building = this.buildings[i];
                if (!building) continue;
                
                this.instancedBuildings.getMatrixAt(i, matrix);
                matrix.decompose(dummy.position, dummy.quaternion, dummy.scale);
                
                // Add subtle rotation
                dummy.rotation.y = building.rotation + Math.sin(time * 0.5 + i) * 0.1;
                
                // Add subtle pulsing to emission
                const pulse = Math.sin(time * 2 + i * 0.1) * 0.1 + 0.3;
                
                dummy.updateMatrix();
                this.instancedBuildings.setMatrixAt(i, dummy.matrix);
            }
            
            this.instancedBuildings.instanceMatrix.needsUpdate = true;
        }
        
        // Update camera position
        updateCamera(time) {
            // Orbital camera movement
            const radius = 25;
            const angle = time * 0.1;
            
            this.camera.position.x = Math.cos(angle) * radius;
            this.camera.position.z = Math.sin(angle) * radius;
            this.camera.position.y = 15 + Math.sin(time * 0.5) * 3;
            
            // Add mouse influence
            this.camera.position.x += this.mousePosition.x * 5;
            this.camera.position.z += this.mousePosition.y * 5;
            
            this.camera.lookAt(0, 5, 0);
        }
        
        // Update performance stats
        updateStats(delta) {
            const fps = Math.round(1 / delta);
            
            // Update modal stats if available
            const fpsElement = document.querySelector('.modal-stats .value');
            if (fpsElement) {
                fpsElement.textContent = `${fps} FPS`;
            }
        }
        
        // Public methods for external control
        setParticleSpeed(speed) {
            this.particleSpeed = speed;
        }
        
        toggleParticles(enabled) {
            if (this.particleSystem) {
                this.particleSystem.visible = enabled;
            }
        }
        
        resetCamera() {
            this.camera.position.set(25, 15, 25);
            this.camera.lookAt(0, 0, 0);
        }
    }

    // Export for global access
    window.CyberCity = CyberCity;

    // Auto-initialize when script loads
    document.addEventListener('DOMContentLoaded', () => {
        console.log('üîß Cyber City script loaded');
    });
